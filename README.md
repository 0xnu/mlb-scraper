MLB Scraper
=============
Scrape www.baseball-reference.com for MLB stats.

Install Requirements
------------

Python 3 is required to the scraper. Install the project's dependencies
with::

    pip3 install -r requirements.txt

Running
-------

Run the scraper::

    python3 baseball.py
