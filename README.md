Scrape www.baseball-reference.com for MLB stats.

## Install Requirements

Python 3 is required to the scraper. Install the project's dependencies with:

```shell
pip3 install -r requirements.txt
```

## Running

Run the scraper::

```shell
chmod +x run.sh && ./run.sh
```
